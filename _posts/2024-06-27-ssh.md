# SSH

## 虚拟机网络配置

NAT模式，设置端口转发

在 NAT 设置窗口中，点击“添加”按钮，填写以下信息：

- **主机端口**：填写一个未被占用的端口，例如 `2222`
- **虚拟机 IP 地址**：填写虚拟机的 IP 地址，例如 `192.168.101.68`
- **虚拟机端口**：填写 `22`（这是 SSH 服务的默认端口）

## 确认 Windows 防火墙允许端口 2222

**打开 Windows 防火墙**：

- 按下 `Win + R` 键，打开运行对话框。
- 输入 `wf.msc` 并按回车键，打开 Windows 防火墙。

**添加入站规则**：

- 在左侧栏中选择“入站规则”。
- 在右侧栏中选择“新建规则”。
- 选择“端口”并点击“下一步”。
- 选择“TCP”，并在“特定本地端口”中输入 `2222`，然后点击“下一步”。
- 选择“允许连接”，然后点击“下一步”。
- 确保所有配置文件（域、专用、公用）都被选中，点击“下一步”。
- 为这个规则命名，例如“SSH Port 2222”，然后点击“完成”。
